# SuperCart Sales Forecast Prediction

Machine Learning web app to predict total sales for SuperKart products.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Docker
- GitHub Actions
- Hugging Face Spaces

## Project Structure

supercart-sales-forecast/
├── Dockerfile
├── deploy.sh
├── requirements.txt
├── .github/workflows/pipeline.yml
└── src/
    ├── streamlit_app.py
    ├── superkart_pipeline.pkl
    └── data/
        ├── SuperKart.csv
        ├── Train.csv
        └── Test.csv
